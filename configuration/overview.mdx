Citizen provides a comprehensive set of configuration variables to customize the appearance and behavior of your MediaWiki installation. All configuration variables use the `$wgCitizen` prefix and should be set in your `LocalSettings.php` file.

## Requirements

Citizen requires MediaWiki 1.43.0 or higher.

## Configuration Structure

Citizen's configuration is organized into several categories:

- **Theme & Appearance** - Control colors, theme defaults, and visual styling
- **Search** - Configure search module, gateway, and behavior
- **Layout** - Control page structure, header position, and responsive features
- **Features** - Enable or disable specific skin features
- **Extensions** - Integration settings for MediaWiki extensions

## Quick Start

Add configuration to your `LocalSettings.php` after loading the skin:

```php
wfLoadSkin( 'Citizen' );

// Example configuration
$wgCitizenThemeDefault = 'dark';
$wgCitizenEnableManifest = true;
$wgCitizenSearchGateway = 'mwRestApi';
```

## Configuration Scope

All Citizen configuration variables are **global** and apply to all users on your wiki. User-specific preferences are handled through MediaWiki's built-in client preferences system.

## Validation

Citizen validates configuration values at runtime. Invalid values will fall back to defaults and may log warnings. Always test configuration changes in a development environment first.
